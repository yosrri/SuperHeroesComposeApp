# Super Heroes Compose App

A modern Android application that displays a collection of superheroes. This project serves as a demonstration of modern Android development practices, built entirely with Kotlin and Jetpack Compose.

## ✨ Features

* **Declarative UI**: Built completely with [Jetpack Compose](https://developer.android.com/jetpack/compose).
* **Material Design 3**: Utilizes Material 3 components and dynamic theming for a modern look and feel.
* **Modern Kotlin**: Leverages Kotlin 2.0+ features and Coroutines/Flows.
* **Dark Theme Support**: Fully supports system-level dark mode toggling.

## 🛠 Tech Stack

This project takes advantage of modern Android development libraries and tools:

* **[Kotlin](https://kotlinlang.org/)** (v2.0.21) - First-class and official programming language for Android development.
* **[Jetpack Compose](https://developer.android.com/jetpack/compose)** (BOM 2024.11.00) - Android's modern toolkit for building native UI.
  * Compose UI
  * Compose Material 3
  * Compose Tooling & Previews
* **[AndroidX](https://developer.android.com/jetpack/androidx)** - Core KTX, Lifecycle Runtime KTX, and Activity Compose.
* **Gradle Version Catalogs** - For centralized dependency management (`libs.versions.toml`).
* **Android Gradle Plugin** (v9.0.1)
